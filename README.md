# online_sales_analysis

## Descrierea proiectului
Acest proiect este un sistem Python pentru analiza datelor de vânzări într-un magazin online. Proiectul folosește concepte de programare orientată pe obiecte (OOP) pentru a gestiona produse, coșul de cumpărături și pentru a calcula valori totale. De asemenea, proiectul este versiovat folosind Git și GitHub, cu ramuri separate pentru funcționalități extinse și gestionarea conflictelor.

## Clase și funcționalități

### Product
- Atribute: `name`, `price`, `quantity`.
- Metode: afișarea informațiilor și actualizarea cantității produsului.

### ProductManager
- Atribut: o listă de produse.
- Metode: adăugarea, afișarea, calcularea valorii totale și eliminarea produselor după nume.

### Cart
- Atribut: lista produselor din coș.
- Metode: adăugarea produselor în coș, afișarea conținutului și calcularea valorii totale a coșului.
